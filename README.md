# Chart n Grid

### Stack
* [webpack] 
* [sass-loader](https://github.com/jtangelder/sass-loader)

### Dev Run
- `npm install`
- `npm install webpack-dev-server -g`
- `webpack-dev-server --progress --colors --inline` will now listen on localhost:8080


[webpack]: <https://webpack.github.io/>
[sass-loader]: <https://github.com/jtangelder/sass-loader>
